# 求职高手 - 后端服务

后端 API 服务，为安卓客户端提供数据接口与 AI Agent 能力。

## 数据获取策略

- 岗位数据：仅采集头部大厂官方招聘网站（首批 15-20 家），不接入第三方招聘平台
- 企业数据：通过天眼查/企查查官方付费 API 获取工商、司法、经营信息
- 合同数据：用户自行上传，不涉及外部采集
- 遵守 robots.txt，合理请求频率，仅采集公开 JD 信息

## 模块规划

### 一、鹰眼猎手（岗位发现与匹配）

**数据源：** 头部企业官方招聘网站（字节、腾讯、阿里、百度、美团、京东、华为、小米等）

**功能：**

1. 岗位采集
   - 定时抓取企业官网公开岗位信息（岗位名称、职责描述、任职要求、工作地点、薪资范围）
   - 每家企业独立解析器，适配不同官网页面结构
   - 增量更新，检测岗位新增/下线/变更

2. 智能匹配
   - 基于用户画像（学历、专业、技能、期望城市、期望薪资）进行多维度匹配
   - AI 分析岗位 JD 与用户简历的匹配度，输出匹配百分比和分析说明
   - 标注新发布岗位，优先推荐时效性强的机会

3. 岗位搜索与筛选
   - 按公司、城市、岗位类型、薪资范围、匹配度筛选
   - 关键词搜索

### 二、幻影投递官（简历定制与投递助手）

**定位：** 投递助手，不自动登录第三方平台，仅操作企业官网公开投递入口

**功能：**

1. 简历定制
   - 用户上传基础简历
   - AI 根据目标岗位 JD 生成定制化简历版本（调整措辞、突出相关经历、优化排版）
   - 用户确认后保存，每个岗位可对应一份定制简历

2. 投递管理
   - 生成投递清单（目标岗位 + 对应定制简历）
   - 用户确认后，通过企业官网公开投递表单代为提交
   - 遇到验证码时交回用户手动处理，不自动绕过
   - 控制投递频率，不批量轰炸

3. 进度追踪
   - 记录每次投递的状态（待投递、已投递、已查看、面试邀约、已拒绝）
   - 投递时间线和统计面板
   - 投递提醒（岗位即将截止、长时间未回复）

### 三、深网调查员（企业风险分析）

**数据源：** 天眼查 API + 企查查 API（官方付费接口）+ 公开新闻舆情

**功能：**

1. 企业基本画像
   - 工商注册信息（成立日期、注册资本、法人、股东结构）
   - 经营状态、行业分类、企业规模
   - 融资历程

2. 风险扫描
   - 司法风险：诉讼记录、被执行信息、失信记录
   - 经营风险：行政处罚、经营异常、税务异常
   - 舆情风险：近期负面新闻（裁员、欠薪、暴雷）

3. AI 风险评估
   - 综合以上数据生成风险等级（低/中/高）
   - AI 输出通俗易懂的风险评估报告
   - 在用户投递/面试/签约前自动触发风险预警

### 四、契约卫士（合同智能解析）

**数据源：** 无外部数据依赖，用户自行上传合同文件

**功能：**

1. 合同解析
   - 支持 PDF / Word / 图片格式上传
   - 图片格式通过 OCR 识别文字
   - 结构化提取合同条款

2. 逐条款风险分析
   - AI 识别关键条款：试用期、薪资构成、竞业限制、违约金、加班规定、社保公积金
   - 每条标注风险等级（安全/警告/危险）
   - 对比劳动法相关规定，标注不合规或对求职者不利的条款

3. 通俗解读
   - 将法律术语翻译为大白话
   - 给出具体建议（"建议协商修改"/"可以接受"/"强烈建议拒绝"）
   - 生成合同整体评估报告（综合评分 + 主要风险点摘要）

### 五、AI 对话服务

**功能：**

1. 统一对话入口
   - 用户通过自然语言提问，系统识别意图后分发到对应模块
   - 流式输出（SSE），实时返回 AI 回复

2. 上下文感知
   - 识别用户问题涉及的模块（岗位咨询 → 鹰眼猎手，企业背景 → 深网调查员 ...）
   - 保持对话上下文，支持多轮追问

3. 通用求职问答
   - 面试技巧、简历建议、薪资谈判等求职知识问答
   - 模拟面试练习
